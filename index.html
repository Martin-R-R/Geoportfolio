My Research Project

I have spent this year working on my honours with Google exploring possible indoor navigation innovations using Cloud Anchors. I have been developing an indoor navigation application in Unity to help navigate the RMIT library to locate books. The app uses the android phone camera to place virtual 'anchors' in augmented reality that are stored in the cloud to be resolved later. Once we build a network of cloud anchors throughout the library we are able to resolve them with arrowed directions displayed over the phone camera to lead the user towards their destination.
